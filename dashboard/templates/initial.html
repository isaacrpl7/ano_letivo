ANO LETIVO

Uma ferramenta para gerenciar escolas, alunos, professores, entre outros elementos, para organizar seu ano letivo!

GERENCIE OS ELEMENTOS DO SEU ANO LETIVO

Precisa cadastrar alunos? Precisa gerenciar os professores? Tem necessidade de saber quais são as disciplinas? Resolva tudo agora!

Icon

ESCOLAS

Cadastre sua(s) escola(s) e gerencie-a(s)! Após isso, você poderá criar novos professores, turmas, alunos, entre outros.

Icon

TURMAS

Após cadastrar uma escola, cadastre suas turmas. Nunca ficou tão fácil poder gerenciar isso!

Icon

ALUNOS

Ao cadastrar uma turma, agora você pode cadastrar alunos e adicionar à turma.

Icon

DISCIPLINAS

Cada turma possui suas disciplinas, portanto, sinta-se livre para cadastrá-las e adicioná-las à turma.

Icon

PROFESSORES

Cada disciplina pode ser ministrada por um professor. Após criá-las, cadastre seus professores!

Icon

HABILIDADES

Cada aluno adquire uma habilidade ao cursar uma disciplina. Elas geram recompensas!
